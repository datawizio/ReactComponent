# Datawiz React Components Library

Welcome to the **Datawiz React Components Library**!
This library provides a collection of reusable and customizable components developed using **React** and **TypeScript**. 
It is designed to help you build interactive and efficient user interfaces for data-driven applications.

### Key features:

- **Reusable Components**: A wide range of modular components that can be easily integrated into your projects.
- **TypeScript Support**: Ensures type safety, enabling developers to catch errors early and improve code quality.
- **Customizable and Scalable**: Built to be flexible and adaptable to various use cases, with props and customization options that make it easy to tailor each component.
- **Storybook Integration**: Explore and interact with all components in a live, visual environment using Storybook. Each component is well-documented with examples and usage guidelines.
- **Ant Design Compatibility**: Built on top of Ant Design's powerful and flexible component system, providing consistency and a proven design language.

### Get started:

To start using the Datawiz.io component library, browse the list of available components on the left-hand side. 
Each component page includes documentation, usage instructions, and live code examples. 
Our React components are built to help you speed up development while maintaining high performance and ease of use.
