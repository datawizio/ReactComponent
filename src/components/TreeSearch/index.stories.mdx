# TreeSearch

Компонент `TreeSearch` cлужит для вывода дерева `Tree` c поиском `Search`. Можно
выводить строку поиска или нет.

Внутри вшит алогритм фильтрации дерева.

## Basic

<Preview withToolbar>
  <Story name="Basic">
    <TreeSearch searchValue="1-" treeData={treeData} height={300} />
  </Story>
</Preview>

  <Story name="With search input">

## With search condition

<Preview withToolbar>
  <Story name="With search condition">
    <TreeSearch
      showSearchInput
      treeData={treeData}
      height={300}
      searchCondition={(value, node) =>
        node.title && node.title.endsWith(value)
      }
    />
  </Story>
</Preview>
