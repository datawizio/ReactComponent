# CheckboxGroup

Компонент `CheckboxGroup` предназначен для вывода группы чекбоксов.

Базовый компонент:
[**antd/checkbox.group**](https://ant.design/components/checkbox/).

## Basic

<Preview>
  <Story name="Basic">
    <CheckboxGroup
      options={[
        { label: "Apple", value: "Apple" },
        { label: "Pear", value: "Pear" },
        { label: "Orange", value: "Orange" }
      ]}
      defaultValue={["Apple"]}
    />
  </Story>
</Preview>
