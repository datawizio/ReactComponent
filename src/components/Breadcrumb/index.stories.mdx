# Breadcrumb

The `Breadcrumb` component provides a way to display a breadcrumb navigation system.  
It is based on the [**Ant Design Breadcrumb**](https://ant.design/components/breadcrumb/) component.

## Props

The `Breadcrumb` component accepts all the props of the [**Ant Design Breadcrumb**](https://ant.design/components/breadcrumb/#API).
