# Tree

Компонент `Tree` cлужит для вывода дерева. Можно выводить просто в виде списка,
можно в виде списка с выбором.

Базовый компонент: [**antd/tree**](https://ant.design/components/tree/).

## Структура данных

```
[
  {
    title: "parent 1",
    key: "0-0",
    children: [
      {
        title: "parent 1-0",
        key: "0-0-0",
        children: [
          {
            title: "leaf",
            key: "0-0-0-0"
          },
          {
            title: "leaf",
            key: "0-0-0-1"
          }
        ]
      }
    ]
  }
]
```

              title: "parent 1-1",

                }

        }
      ];

    }}
